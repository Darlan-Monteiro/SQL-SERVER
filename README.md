# SQL SERVER

🗄️ Roadmap e Códigos de Estudo
Este repositório documenta a minha jornada prática de estudos em SQL Server. Aqui você encontrará todos os scripts, exercícios e anotações, organizados de acordo com a minha evolução no banco de dados, indo desde consultas básicas até integrações complexas com Python e Power BI.

🎯 Trilha de Conhecimento e Progresso
O mapeamento abaixo reflete os tópicos abordados. Os links para os códigos e pastas estão sendo adicionados conforme o avanço dos estudos.

Fase 1: Fundamentos e Consultas Básicas
- Instalação e Configuração: Configuração do SQL Server, SSMS e Restauração do BD Contoso.
- Introdução ao SQL: Estrutura básica de consultas.
- Ordenação e Filtragem: Uso de WHERE, AND, OR, NOT, IN, BETWEEN, LIKE e IS NULL.
- Funções de Agregação: Aplicação de SUM, AVG, COUNT, MAX e MIN.
- Agrupamentos: Uso prático do GROUP BY.

Fase 2: Relacionamentos
- JOINs: Implementação de INNER, LEFT, RIGHT, FULL e ANTI JOINs.
- Aplicações Práticas: Combinação avançada de GROUP BY com JOINs.

Fase 3: Lógica e Manipulação de Dados
- Variáveis: Declaração e uso em scripts.
- Strings e Datas: Funções para manipulação e formatação.
- Funções Condicionais: Uso de CASE WHEN, IIF e COALESCE.
- SQL Views: Criação e gerenciamento de exibições virtuais.
- CRUD: Manipulação direta de dados via INSERT, UPDATE e DELETE.

Fase 4: Consultas Avançadas
- Subqueries e CTE's: Expressões de Tabela Comuns e subconsultas para lógicas complexas.
- Loops no SQL: Estruturas de repetição com WHILE.
- Window Functions: Funções de Janela como ROW_NUMBER, RANK, LEAD e LAG.
- Regex (Expressões Regulares): Busca de padrões avançados em texto.

Fase 5: Arquitetura e Automação
- Constraints: Restrições para garantir a integridade dos dados.
- Sequences: Geração de valores sequenciais.
- Transactions: Controle de transações com BEGIN TRAN, COMMIT e ROLLBACK.
- Functions e Procedures: Criação de funções e procedimentos armazenados.
- Triggers DML e DDL: Gatilhos automatizados para eventos no banco.
- Pivot Table: Transformação de linhas em colunas para relatórios.

Fase 6: Projetos e Integrações Práticas
- Projeto 1: Integração SQL Server e Excel.
- Projeto 2: Integração SQL Server e Power BI.
- Projeto 3: Integração SQL Server e Python.

🛠️ Como navegar neste repositório
- Cada tópico acima corresponde a um arquivo .sql ou uma pasta específica.
- Os scripts contêm comentários práticos (passo a passo) explicando a lógica utilizada.
- Os projetos finais estão em pastas isoladas com os scripts SQL e os arquivos das ferramentas (ex: .pbix ou .ipynb).
